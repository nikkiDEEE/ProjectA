# ProjectA

## Dhoom 2 Inspired "A" Insignia - 3D Render

A 3D render of the iconic "A" insignia from Dhoom 2, inspired by the signature that Hrithik Roshan's character (Mr. A) leaves behind in the movie.

### Features

- **3D Rotating Animation**: The golden "A" continuously rotates in 3D space
- **Metallic Golden Finish**: CSS gradients create a realistic golden metallic look
- **Interactive Controls**: 
  - Toggle rotation on/off
  - Toggle glow effect
  - Manual rotation via mouse drag (when auto-rotation is paused)
- **Keyboard Shortcuts**:
  - `SPACE` - Toggle rotation
  - `G` - Toggle glow effect
- **Sparkle Effects**: Click anywhere on the scene to create sparkle particles
- **Responsive Design**: Works on desktop and mobile devices

### Usage

Open `index.html` in a web browser to view the 3D insignia.

### Technologies Used

- HTML5
- CSS3 (3D transforms, animations, gradients)
- Vanilla JavaScript
